# MJ GT Mei 2026

Repositori ini berisi dokumen naskah MJ GT untuk bulan Mei 2026 dalam format Markdown.

## Struktur Repositori

- `2026-05/` berisi naskah per tanggal/minggu.
- `mj gt mei 2026.pdf` merupakan berkas referensi lokal dan diabaikan oleh git melalui `.gitignore`.

## Konvensi Penamaan

Seluruh berkas di `2026-05/` menggunakan pola:

`YYYY-MM-DD-judul-dalam-huruf-kecil-dengan-strip.md`

Contoh:

- `2026-05-03-melekat-seutuhnya-pada-tuhan.md`
- `2026-05-31-allah-menghendaki-harmoni.md`

## Susunan Isi Dokumen

Setiap dokumen umumnya memuat bagian berikut:

- judul liturgi atau minggu gerejawi
- tema
- bacaan Alkitab
- tujuan jemaat
- pembahasan
- arah renungan

## Petunjuk Penggunaan

- Buka berkas Markdown sesuai tanggal atau tema yang diperlukan.
- Gunakan PDF sebagai pembanding apabila diperlukan.
- Saat menambah naskah baru, ikuti pola penamaan yang sama agar konsisten dan mudah ditelusuri.
